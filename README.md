# Vinesh Reddy Kankanalapally — Portfolio

> Personal portfolio website built with plain HTML, CSS & JavaScript. Inspired by [Brittany Chiang](https://brittanychiang.com).

🌐 **Live site:** [vineshreddyk.github.io](https://vineshreddyk.github.io)

---

## Features

- Two-column layout — fixed left panel, scrollable right panel
- Active nav highlighting on scroll
- Cursor spotlight glow effect
- Fully responsive (mobile friendly)
- Sections: About, Experience, Projects, Certifications, Education

## Built With

- HTML5
- CSS3 (custom properties, CSS Grid, Flexbox)
- Vanilla JavaScript

## Running Locally

No build step needed — just open `index.html` in your browser.

```bash
git clone https://github.com/VineshReddyK/VineshReddyK.github.io.git
cd VineshReddyK.github.io
open index.html
```
